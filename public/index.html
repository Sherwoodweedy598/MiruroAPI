Streaming (3-Step Flow)
How streaming works: To get a video URL, follow these 3 steps in order. Each step's output feeds into the next.
1GET /api/episodes/{anilist_id}
Get all episodes from multiple providers (kiwi, bee, bonk, ally, moo, etc.) organized by sub/dub. Returns episode IDs for Step 2.
{
  "providers": {
    "kiwi": { "episodes": { "sub": [{ "id": "watch/kiwi/20/sub/animepahe-1", "number": 1, ... }] } },
    "bee": { ... },
    "bonk": { ... }
  }
}
Try it (Naruto)
2 /api/watch/{provider}/{anilistId}/{category}/{slug} RECOMMENDED
Take the id from Step 1 and use it as the URL. Returns M3U8 streams, subtitles, intro/outro timestamps.
{
  "streams": [
    { "url": "https://.../master.m3u8", "type": "hls", "quality": "1080p" }
  ],
  "subtitles": [ { "file": "...", "label": "English" } ],
  "intro": { "start": 0, "end": 90 },
  "outro": { "start": 1300, "end": 1420 }
}
Try it
3 Play the stream
Feed streams[0].url into any HLS player (Video.js, hls.js, VLC, mpv). Use intro/outro timestamps for skip buttons.
All collection endpoints return: { page, perPage, total, hasNextPage, results[] }
